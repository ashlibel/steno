What is it?

Code at
steno speed

STENO is an interpreted language with a built-in transpiler to Python. Every keyword is a 2–3 character abbreviation — lp for loop, fn for function, pr for print. Just like a stenographer who writes entire words in one chord, STENO lets you write entire constructs in a single token.

in Python:

for i in range(0, 10):
    print(i)

def add(a, b):
    return a + b

in STENO:

lp i .. 0 10
    pr i

fn add a b
    rt a + b

calculator.sten 
# fn, rt, tr/ex, if/el

fn divide a b
    tr
        if b == 0
            pr "Error: division by zero"
            rt None
        rt a / b
    ex Exception e
        pr "Unexpected:" e

pr divide(10, 2)   # 5.0
pr divide(9, 0)    # error
data_structures.sten 
# cl, fn, self, methods

cl Stack
    fn __init__ self
        self.items = []
        self.size = 0

    fn push self item
        self.items.append(item)
        self.size += 1

    fn pop self
        if self.size == 0
            pr "Stack underflow!"
            rt None
        self.size -= 1
        rt self.items.pop()

vr s = Stack()
lp i .. 1 6
    s.push(i * 10)
pr "Pop:" s.pop()

# Caesar cipher — fn, lp, string ops

fn caesar_encrypt text shift
    vr result = ""
    lp ch text
        vr code = ord(ch)
        if code >= 65 and code <= 90
            result += chr((code-65+shift) % 26 + 65)
        el if code >= 97 and code <= 122
            result += chr((code-97+shift) % 26 + 97)
        el
            result += ch
    rt result

vr enc = caesar_encrypt("Hello STENO", 13)
pr enc   # Uryyb FGRAB

# Fibonacci — fn, if, rt, lp, pr

fn fibonacci n
    if n <= 1
        rt n
    rt fibonacci(n - 1) + fibonacci(n - 2)

lp i .. 0 10

Under the hood

Architecture

A classic pipeline: lex → parse → interpret or transpile.

your_file.sten │ ▼ lexer.py Token stream (TT enum, Token dataclass) │ ▼ parser.py AST (ast_nodes.py dataclasses) │ ├──▶ interpreter.py tree-walk, direct execution │ └──▶ transpiler.py AST → Python source steno.py wires everything — CLI + REPL
Get started

Up & running
in 60 seconds

1.

Clone the repo

git clone https://github.com/ashlibel/steno.git cd steno
2.

Run the demo

No dependencies beyond Python 3. Runs all five example programs at once.

python steno.py demo
3.

Run a single file

python steno.py run hello_world.sten
4.

Open the REPL

Interactive prompt — type STENO one line at a time.

python steno.py
5.

Debug & transpile

python steno.py tokens fizzbuzz.sten # token stream python steno.py ast fizzbuzz.sten # AST dump python steno.py check fizzbuzz.sten # syntax only python steno.py compile fizzbuzz.sten # → Python
Real steno

Write STENO
with actual steno

STENO ships with a Plover dictionary so you can chord the language on a real steno keyboard. Follow these three steps to get set up:

1.

Download & install Plover

Plover is the free, open-source steno engine. Grab the latest release for your OS.

openstenoproject.org/plover/ →
2.

Build your steno keytoppers

Turn your regular keyboard into a steno machine with printed keytoppers — free and printable. Must be used with an Nkey rollover keyboard.

cemrajc.github.io/stenotoppers/ →
3.

Load the STENO dictionary

In Plover's dictionary manager, add steno_final_dict.json from the repo above your main dictionary. That's it — start chordin'.
